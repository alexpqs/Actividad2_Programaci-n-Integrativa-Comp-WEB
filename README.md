- `components/`: Contiene todos los archivos JavaScript para los componentes personalizados.
- `data/`: Contiene datos JSON opcionales para la galería y otros componentes.
- `index.html`: Archivo principal que integra todos los componentes.
- `styles/`: Estilos globales y recursos compartidos.
- `README.md`: Este archivo con información de uso.

## Instrucciones
1. Clonar este repositorio.
2. Abrir `index.html` en un navegador.
3. Verificar el funcionamiento de los componentes y la navegación.

## Componentes
### Header
Encabezado con un título estilizado.
### Footer
Pie de página con información de copyright.
### Main
Contenedor principal que utiliza slots para cargar páginas dinámicamente.
### Menu
Menú de navegación funcional.
### SocialProfile
Página de perfil con información estática del usuario.
### CustomTable
Tabla que carga datos desde la API: `https://jsonplaceholder.typicode.com/users`.
### Gallery
Galería que carga imágenes desde una API de tu elección (por defecto `https://pokeapi.co/`).
